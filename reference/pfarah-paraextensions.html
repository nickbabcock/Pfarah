collect prop obj
Signature: prop:string -> obj:ParaValue -> ParaValue []

Finds all the properties of the object with a given key and aggregates all the values under a single array.
tryFind prop obj
Signature: prop:string -> obj:ParaValue -> (string * ParaValue) option

Tries to find the first property of the object that has the given key. If a property is found then Some ParaValue will be returned else None
